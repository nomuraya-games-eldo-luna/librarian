# シーン生成プロンプトテンプレート

**用途**: 新しいシーンを執筆する際の標準プロンプト

**最終更新**: 2025-12-27

---

## 📋 プロンプト本文

```
【シーン執筆依頼】

Part{部} Chapter {章} の Scene {シーン番号} を執筆してください。

## 前提確認

以下を必ず読んでから執筆を開始してください:

1. **品質基準**: `設定メモ（簡易版）.md`
2. **引き継ぎ要件**: `{該当する引き継ぎ書ファイル名}`
3. **前章**: `docs/part{部}/chapter-{章-1}.md`
4. **現在の章**: `docs/part{部}/chapter-{章}.md`（既存部分）

## シーン情報

### シーンの目的
- {このシーンで達成すべきこと}
- {次のシーンへの繋ぎ}

### 登場人物
- {主要キャラクター名とその状態}

### 場所・時間
- {場所}: {場所の説明}
- {時間帯}: {時刻・天候等}

### 重要な制約
- {このシーンで避けるべきこと}
- {このシーンで必ず含めるべき要素}

## 執筆の流れ

### ステップ1: プランニング
まず、以下を提案してください:
- シーンの展開（起承転結）
- 重要な会話・描写のポイント
- 文字数見込み（4000-6000文字目安）

### ステップ2: 初稿生成
合意形成後、以下を守って執筆:
- アーティファクトで全文生成
- 没入感阻害要素の完全排除（章番号・見出し・現実表現）
- 五感描写を重視（視覚・聴覚・触覚・嗅覚・味覚）
- 映画的な臨場感

### ステップ3: セルフレビュー
生成後、以下をチェック:
- [ ] メタ情報の排除（章番号・見出し等）
- [ ] 場面転換が自然（つぎはぎ感なし）
- [ ] 五感描写が豊富
- [ ] 比喩表現が詩的
- [ ] センシティブ要素の品質（該当シーンのみ）
- [ ] 利用規約遵守

### ステップ4: 改稿提案
問題があれば、具体的な修正案を提示。

## 品質基準（再掲）

### 没入感阻害要素の完全排除
❌ NG: 「第3章」「3章」「Scene 1」等のメタ情報
❌ NG: 見出し（`##` や `###`）
❌ NG: 「三日前」「1週間」→ ✅ OK: 「ここ数晩」「早めの対処を希望」

### 五感描写の重視
- **視覚**: 場面が目に浮かぶレベルの詳細描写
- **聴覚**: 音響効果、声のトーン、環境音
- **触覚**: 肌感覚、温度、質感
- **嗅覚**: 匂いによる情景・感情表現
- **味覚**: 食事・状況に応じた味覚描写

### センシティブ要素（該当シーンのみ）
- 目的・意味重視（マンネリ回避）
- 丹念な描写（あっさり終わるのはNG）
- 利用規約遵守（詩的・暗示的表現）
```

## 🔄 使用例

```markdown
【シーン執筆依頼】

Part2 Chapter 10 の Scene 1 を執筆してください。

## 前提確認

以下を必ず読んでから執筆を開始してください:

1. **品質基準**: `設定メモ（簡易版）.md`
2. **引き継ぎ要件**: `2-3以降執筆引き継ぎ書.md`
3. **前章**: `docs/part2/chapter-9.md`
4. **現在の章**: `docs/part2/chapter-10.md`（まだ未作成）

## シーン情報

### シーンの目的
- 盗賊討伐から帰還し、ギルドに報告
- エルドとルナの「ぎこちなさ」を継続
- 次の依頼への伏線（魔法覚醒）

### 登場人物
- エルド: 戦闘疲労、ルナへの罪悪感で目を合わせられない
- ルナ: エルドの変化に気づいている、内心で煩悶
- ミラ（ギルド受付）: 二人の様子に気づくが深く追及しない

### 場所・時間
- 場所: ギルド本部（受付カウンター）
- 時間帯: 夕刻、夕日が差し込む

### 重要な制約
- エルドがルナに「見ていた」ことを告白しない（禁止事項）
- ルナがエルドに気づいていることを明かさない（禁止事項）
- 読者だけが両方の真実を知っている緊張感を維持

## 執筆の流れ

（以下、テンプレート通り）
```

## 📝 カスタマイズポイント

各シーンに応じて、以下をカスタマイズ:

1. **シーンの目的** - 物語上の役割
2. **登場人物** - 状態・感情
3. **場所・時間** - 雰囲気・天候
4. **重要な制約** - 禁止事項・必須要素

---

**このテンプレートを使うことで、品質基準を満たしたシーンを効率的に生成できます。**
