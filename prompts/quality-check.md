# 品質チェックプロンプトテンプレート

**用途**: 執筆後のセルフレビューを標準化

**最終更新**: 2025-12-27

---

## 📋 プロンプト本文

```
【品質チェック依頼】

以下のシーンを品質基準に照らしてレビューしてください。

## レビュー対象

{生成したシーン本文をここに貼り付け}

---

## チェック項目

### 1. 没入感阻害要素の排除

**確認内容**:
- [ ] 章番号言及（「第3章」「3章」等）がないか
- [ ] 見出し（`##`, `###`）が含まれていないか
- [ ] 現実表現（「三日前」「1週間」「24時間」等）がないか

**NG例**:
- 「第3章で起きた事件を思い出し...」
- 「## 戦闘シーン」
- 「三日前に受けた依頼」

**OK例**:
- 「あの事件を思い出し...」
- （見出しなしで場面転換）
- 「ここ数晩の依頼」

**結果**: ✅ 問題なし / ❌ 問題あり（具体的に指摘）

---

### 2. 文章品質

**確認内容**:
- [ ] 文字数は4000-6000文字目安に達しているか
- [ ] 場面転換が自然か（つぎはぎ感がないか）
- [ ] 一貫した流れで構成されているか

**結果**: ✅ 問題なし / ⚠️ 改善推奨（具体的に指摘）

---

### 3. 五感描写の充実度

**確認内容**:
- [ ] **視覚**: 場面が目に浮かぶレベルの詳細描写があるか
- [ ] **聴覚**: 音響効果、声のトーン、環境音が描写されているか
- [ ] **触覚**: 肌感覚、温度、質感の表現があるか
- [ ] **嗅覚**: 匂いによる情景・感情表現があるか（任意）
- [ ] **味覚**: 食事シーンがあれば味覚描写があるか（任意）

**具体例の抽出**:
- 視覚: 「{該当箇所を引用}」
- 聴覚: 「{該当箇所を引用}」
- 触覚: 「{該当箇所を引用}」

**結果**: ✅ 充実 / ⚠️ やや不足（追加推奨） / ❌ 不足（要改善）

---

### 4. 映画的描写

**確認内容**:
- [ ] 読者が「映画を見ている気分」になるレベルの臨場感があるか
- [ ] 比喩表現が詩的で美しいか
- [ ] 心理描写が行動の背景にある感情・意図を細かく表現しているか

**優れた表現の抽出**:
- 「{詩的な比喩表現を引用}」
- 「{心理描写の好例を引用}」

**結果**: ✅ 映画的 / ⚠️ やや平坦（表現の工夫を推奨）

---

### 5. センシティブ要素の品質（該当シーンのみ）

**該当する場合のみチェック**:
- [ ] 目的・意味重視（単なる繰り返しではない）
- [ ] マンネリ回避（シチュエーション・タイミングに新鮮さ）
- [ ] 説明禁止（「二人の愛情表現」等のメタ説明がない）
- [ ] 丹念な描写（あっさり終わっていない）

**結果**: ✅ 問題なし / ⚠️ 改善推奨 / N/A（該当シーンなし）

---

### 6. 利用規約遵守

**確認内容**:
- [ ] 性器の露骨な描写を回避
- [ ] 暴力的・強制的要素を回避
- [ ] 未成年を想起させる要素を回避
- [ ] 詩的・芸術的な暗示表現で美的に描写

**結果**: ✅ 遵守 / ❌ 違反の可能性（具体的に指摘し、代替案提示）

---

### 7. ストーリー構成

**確認内容**:
- [ ] 勢い重視でダレ場がないか
- [ ] 緊張感が継続しているか
- [ ] 読者が続きを求める引力があるか

**結果**: ✅ 良好 / ⚠️ やや冗長（カット推奨箇所を指摘）
```

## 🔄 使用例

```markdown
【品質チェック依頼】

以下のシーンを品質基準に照らしてレビューしてください。

## レビュー対象

（執筆したシーン本文をここに貼り付け）

エルドとルナは、夕暮れ時のギルド本部に足を踏み入れた。
受付カウンターでは、いつものミラが書類整理をしている。
「お帰りなさい。盗賊討伐、お疲れ様でした」
ミラの声は、いつもの明るさを保っていたが、エルドは視線を逸らした。
...（続く）

---

（以下、チェック項目に従ってレビュー）
```

---

## 📝 カスタマイズポイント

シーンの性質に応じて、重点チェック項目を調整:

- **戦闘シーン**: 五感描写・緊張感を重点チェック
- **会話シーン**: 心理描写・キャラクターの一貫性を重点チェック
- **センシティブシーン**: 利用規約遵守・丹念な描写を最重点チェック

---

**このテンプレートで品質を客観的に評価し、改善点を明確化できます。**
